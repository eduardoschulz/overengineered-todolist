# overengineered-todolist
An overengineered todo list


# Development Guide
## Creating your dev environment
```bash
python3 -m venv .venv

# Linux and MacOS
source .venv/bin/activate

#Windows
.venv\Scripts\activate.bat
```

### Installing dependencies
```
python3 -m pip install -r requirements.txt
```
